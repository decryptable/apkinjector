APK Injector
============

APK Injector is a pluginable tool designed to automate the injection of libraries into APKs. It supports unpacking, repacking, and handling of bundles.

Visit our GitHub repository: `APK Injector GitHub Repository <https://github.com/nitanmarcel/apkinjector>`_
